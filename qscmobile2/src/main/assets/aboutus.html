关于求是潮 Mobile

求是潮 Mobile 2.0 是求是潮的新一代移动客户端，提供 课表 / 成绩 / 考试 / 校车 等方便学生的查询。

下阶段将提供校内活动通知，开发者开放平台，敬请关注！

Mobile 由求是潮技术研发中心出品，产品运营部门提供运营支持。

What's New

  • 卡片式主界面

  • 增加周课表视图

  • 新的校历视图

  • 更优雅流畅的动画

  • 全手势操作支持

  • 使用 HTTPS 以提升安全性

  • 修复课表，校车，校历的一些 Bug
